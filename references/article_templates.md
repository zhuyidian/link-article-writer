1. [科技评测](#科技评测) - 产品体验、横向对比、选购建议

## 科技评测

### 适用场景

- 单品评测：深入体验某款产品
- 横评对比：多款产品同维度对比
- 选购指南：帮助用户做购买决策

### 文章结构

```markdown
# [产品名称] 评测：[核心结论]

## 开场（HOOK）
[反常识开头/场景代入/核心亮点预告]

## 产品概览
- 基本参数
- 适用人群
- 首发价格/现价

## 核心体验
### [功能点1]
[体验描述 + 使用场景]

### [功能点2]
...

## 优缺点分析
### 优点
1. 优点1
2. 优点2

### 缺点
1. 缺点1
2. 缺点2

## 竞品对比（如适用）
| 维度 | 本品 | 竞品A | 竞品B |
|------|------|-------|-------|
| 价格 | | | |
| 性能 | | | |
| 续航 | | | |

## 总结建议
[适合谁买/不适合谁买]

## 相关资源
- [官网链接]
- [购买链接]
```

| 档位 | 字数 | 插图数量 |
|------|------|---------|
| 快速体验 | 800-1200 | 2-3张 |
| 标准评测 | 2000-3000 | 4-5张 |
| 深度横评 | 4000-6000 | 6-8张 |

- 行业趋势分析
- 技术原理探秘

- 数据解读

## 现象/事件概述
[是什么引起了关注？]

## 背景与原因

[直接导火索]

### 深层原因
[背后驱动因素]

[技术层面的分析]

[商业模式、盈利方式]

### 用户视角
[用户需求、使用场景]

### 行业视角
[对行业的影响]

## 关键数据
| 数据点 | 数值 | 说明 |
|--------|------|------|
| | | |

## 各方反应
### 支持者
[观点摘录]

### 质疑者
[观点摘录]

### 中立观察
[第三方观点]

- 短期趋势

- 长期趋势

## 教程指南

### 适用场景

- 操作步骤讲解
-入门科普
- 疑难解答

### 文章结构

```markdown
# [任务/目标]：从零开始的完整指南

## 问题场景
[读者遇到的困惑/需要解决的问题]

## 解决方案概览
[先给全局认知]

## 准备阶段
### 所需工具/账号
### 环境要求

## 具体步骤
### 第一步：[具体操作]
[详细说明]

### 第二步：[具体操作]
...

### 第三步：[具体操作]
...

## 常见问题（FAQ）
### Q1: [问题]
A: [解答]

### Q2: [问题]
A: [解答]

## 进阶技巧
[可选：给中级用户的高级提示]

## 总结
[回顾关键要点]
```

### 字数参考

| 档位 | 字数 | 插图数量 |
|------|------|---------|
| 快速上手 | 800-1500 | 3-4张 |
| 完整指南 | 2000-3500 | 5-7张 |
| 疑难解答 | 3000-5000 | 6-8张 |

### 写作技巧

- 每个步骤都有明确的目标
- 复杂操作配合截图说明
- 关键步骤加粗或高亮
- 提供快捷键（如适用）

## 热点评论

### 适用场景

- 突发新闻评论
- 事件解读
- 观点输出
- 舆论回应

### 文章结构

```markdown
# [事件名称] 事件解读：[态度/立场]

## 事件概述
[简要说明发生了什么]

## 事件时间线
- [时间点1]
- [时间点2]
- [时间点3]

## 各方反应
### 当事人/官方
[表态摘录]

### 专家观点
[专业分析]

### 网友热议
[舆论走向]

## 深度解读
### 为什么重要
[事件的意义]

### 背后逻辑
[深层原因分析]

### 可能影响
[对xxx的影响]

## 反思与讨论
[开放性问题]

## 结语
[留下思考空间]
```

### 字数参考

| 档位 | 字数 | 插图数量 |
|------|------|---------|
| 快评 | 600-1000 | 1-2张 |
| 标准评论 | 1500-2500 | 2-3张 |
| 深度解读 | 3000-4500 | 3-4张 |

### 写作注意事项

- 开头直接切入事件
- 观点明确，不含糊
- 提供多元视角
- 结尾留有余韵

## 通用写作规范

### 开场（HOOK）写法

| 类型 | 示例 |
|------|------|
| 反常识开头 | "90%的人都不知道..." |
| 场景代入 | "想象这样一个场景..." |
| 尖锐问题 | "为什么同样是..." |
| 数据冲击 | "仅用3个月，..." |
| 故事开头 | "这一切始于..." |

### 结尾（CLOSE）写法

| 类型 | 适用场景 |
|------|---------|
| 回扣开头 | 首尾呼应 |
| 开放问题 | 引发思考 |
| 视野拉远 | 更高维度总结 |
| 个人感悟 | 情感共鸣 |

### 金句位置

- 文章开头后（吸引继续阅读）
- 每个section结尾（承上启下）
- 文章结尾前（高潮收束）

### 禁止清单

```
✗ "众所周知..."
✗ "让我们来看看..."
✗ "首先...其次...最后..."
✗ "这是一个革命性的技术"
✗ "简单来说..."
✗ 过度感叹号
✗ 堆砌专业术语
✗ AI写作痕迹（检查清单）
```
